# App Base

_Base to start building react js project_

## Starting 🚀

_First of all see different branches to see what is the last version._

### Pre-requisites 📋

_Node JS_
_Code editor (ex: Visual Studio Code)_

### Instalation 🔧

**1: cd /app-base**
**2: yarn install**
**3: yarn start**
**Otherwise you can use NPM Scripts**

## Testing ⚙️
